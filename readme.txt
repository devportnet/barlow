== Theme Setup ==

= Installation =

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Barlow Column Plugin Requirements ==

Barlow has extended support for WooCommerce: https://wordpress.org/plugins/woocommerce

== Setup ==

Barlow requires only content and your skills to turn it into a better theme. It is a simple theme that works great for personal blogs.

== Social Navigation ==

Barlow does support social links in the footer. To add them, simply create a new menu in your WordPress dashboard, add custom links to your profile, for example: https://twitter.com/. Be sure to choose the 'social' display option and save any changes. Our themes support most popular social networks.
